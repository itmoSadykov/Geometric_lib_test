# Math formulas
## Area
- Circle: S = πR²
- Rectangle: S = ab
- Square: S = a²

## Perimeter
- Circle: P = 2πR
- Rectangle: P = 2a + 2b
- Square: P = 4a

## Общее описание решения

Geometric Lib - это библиотека на Python для вычисления площадей и периметров различных геометрических фигур. 
Библиотека предоставляет простой и интуитивно понятный интерфейс для математических вычислений, связанных с геометрией.

Библиотека поддерживает следующие фигуры:
- Круг
- Квадрат  
- Прямоугольник
- Треугольник

## Описание функций

### Модуль circle.py

#### area(r)
   Параметры:
           r (float): радиус круга

   Возвращаемое значение:
           area (float): площадь круга

```python
area(5)
78.53981633974483
```


#### perimeter(r)
Вычисляет длину окружности.

       Параметры:
               r (float): радиус окружности

       Возвращаемое значение:
               perimeter (float): длина окружности
```python
perimeter(5)
31.41592653589793
```

### Модуль square.py

#### area(a)
Вычисляет площадь квадрата.

       Параметры:
               a (float): длина стороны квадрата

       Возвращаемое значение:
               area (float): площадь квадрата
```python
area(4)
16
```

#### perimeter(a)
Вычисляет периметр квадрата.

       Параметры:
               a (float): длина стороны квадрата

       Возвращаемое значение:
               perimeter (float): периметр квадрата
```python
perimeter(4)
16
```

### Модуль rectangle.py

#### area(a, b)
Вычисляет площадь прямоугольника.

       Параметры:
               a (float): длина первой стороны
               b (float): длина второй стороны

       Возвращаемое значение:
               area (float): площадь прямоугольника
```python
area(3, 4)
12
```

#### perimeter(a, b)
Вычисляет периметр прямоугольника.

       Параметры:
               a (float): длина первой стороны
               b (float): длина второй стороны

       Возвращаемое значение:
               perimeter (float): периметр прямоугольника
```python
perimeter(3, 4)
14
```

### Модуль triangle.py

#### area(a, h)
Вычисляет площадь треугольника.

       Параметры:
               a (float): длина основания треугольника
               h (float): высота треугольника

       Возвращаемое значение:
               area (float): площадь треугольника
```python
area(6, 4)
12.0
```

#### perimeter(a, b, c)
Вычисляет периметр треугольника.

       Параметры:
               a (float): длина первой стороны
               b (float): длина второй стороны
               c (float): длина третьей стороны

       Возвращаемое значение:
               perimeter (float): периметр треугольника
```python
perimeter(3, 4, 5)
12
```
